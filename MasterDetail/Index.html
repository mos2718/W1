Master-detail 設計主從網頁

  • master detail 簡介
  • JS window.open 開啟子網頁
  • cookie 跨網頁傳遞資料

程式範例

  • 自己設計的 JSON 物件列表(含觸發事件)
  • FHIR JSON JS 物件列表(含觸發事件)

 

實習題目六: 應用 master detail

配合你的課程專題， master detail 網頁，達成以下功能要求:

  • 取得實習五上傳的 JSON 資料，設計專題所需列表網頁(須包含觸發事件)。例如: 病患列表或貨物清單網頁。個人專題需求設計網頁表單
  • 點選網頁列表後，將選擇的資料寫入 cookie，並開啟你設計的下一個網頁(detail) 網頁。可單純呈現選擇項目之細部內容，或呈現進一步填寫的表單(如醫護表單或該貨品訂單。
