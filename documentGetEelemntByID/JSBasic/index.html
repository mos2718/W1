取得或設定 HTML 標籤內容


取得或設定排版標簽包含的內容

取得或設定段落(p), 標題( h*), 表格(table 中td) 等排版標籤的內容. 語法:
  1. document.getElementById("tagID").innerHTML = "<a href='URL'>要連結的網址</a>"; //指定其內含的 HTML 內容
  2. document.getElementById("tagID").innerTEXT = "Hello world";  // 指定其內含之純文字內容
範例:
設定 p 標籤內容
設定 table 標籤內容: 99 乘法表

取得或設定標簽的屬性

可設定或任何標籤之屬性(但須先知道標籤有哪些屬性, 可請教 Google 大神! 語法:
  1. document.getElementById("imgTagID").src ="***.gif"; // 設定img標籤的圖片來源
  2. document.getElementById("tagID").style.fontSize = "50pt"; // 設定所包含內容之字體大小
範例:
設定 p 標籤包含之文字字形大小屬性
利用滑鼠移入移出事件，設定影像標籤 src 來源屬性。
設定 body bgcolor 顏色
加法程式，取得即設定 text input value。需進行輸入型態檢查(防呆:foolproof)，確保輸入為數值
