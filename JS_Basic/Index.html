實習題目四: JS 基礎指令與 HTML 表單配合應用

在你的醫護表單當中，撰寫程式，達成以下功能要求::

  • 設定陣列變數，並給定資料。開啟網頁時，執行 JS 程式，將陣列中的資料加到下拉選單當中。(注意 JS 程式放置位置)
  • 新增身高體重輸入欄位，以及 BMI 呈現標籤，並新增一個 button 按鈕，點選按鈕後，計算 BMI數值，並在網頁中呈現。並配合性別輸入，判斷 BMI 值是否過高。若過高，網頁出現警告訊息(如:哇!超級肥)。(網頁可增加防呆:foolproof 功能，確保輸入之身高體重一定是數值)。 BMI 參考網址: http://health99.hpa.gov.tw/OnlinkHealth/Onlink_BMI.aspx 
  • 網頁表單當中新增一個 table 及 button 按鈕，點選 button 後，抓取表單當中所有 checkbox 欄位，並將每一個被勾選(checked) 的 checkbox 之 id 及 value 資訊寫入陣列當中。最後取出陣列當中之表單勾選資訊，填入 HTML table 當中。
