標準化醫護表單建構步驟(發展前端網頁或 APP 表單，與 FHIR server 整合應用)

 

產生及上傳 FHIR data

  1. 依據醫療健康照護情境，取得 FHIR server 上病人、就醫資訊
  2. 設計 HTML FORM 輸入表單(表單模板)
  3. 確立 HTML FORM 與 FHIR resource 對應
  4. JS 程式讀取表單內容，轉換成 FHIR XML or JSON data
  5. HTTP　post data to FHIR server

 

查詢及呈現 FHIR server 上的表單資料

  1. 依據醫療健康照護情境，取得 FHIR server 上病人、就醫資訊
  2. 搭配病人、就醫、時間等條件，使用 FHIR API 查詢 FHIR server 上的表單資料
  3. JS 解析 FHIR XML or JSON data
  4. 使用 JS 將 FHIR data 排版呈現(HTML 版面)
