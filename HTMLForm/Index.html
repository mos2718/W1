實習題目二:多樣之醫護表單

健康醫療會使用到許多醫護表單，發展醫護網頁表單，可減輕許多醫護人員的負擔，並促進健康醫療的效益與效率。 同學可挑戰設計各類網頁醫護表單，利於病人或醫護人員利用瀏覽器網頁，填寫健康醫療資訊。


本次練習僅設計前端 HTML Form 網頁(含網頁前後端程式連結設定)。

要求:

  • 找到一張別人沒做過的醫護表單。可與熟悉之醫護人員討論，或上網搜尋。找到後依其分類，將表單圖片發佈到 moodle 實習題目二醫護表單。先選先贏。
  • 醫護表單通常欄位眾多，可僅設計原參考表單之部分內容即可，但須至少包含單選、多選、下拉選擇等 HTML Form input 標籤。
  • 參考原醫護表單版面，整齊排列網頁內容。可用 table、div 等標籤來協助排版。
  • 參考此表單範例，在 form 標籤設定 server 端接收程式。以利按下 button 後，將填寫資料上傳到 web server。 同學可觀察表單 input 標籤中 name and value 屬性，與上傳後回應結果之對應。
