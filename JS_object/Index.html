實習題目五: 設計及處理 JSON data

配合你的課程專題，設計資料存取的 JSON data，參考 JSON 資料互通範例程式，達成以下功能要求:

  • 依據個人專題需求設計網頁表單
  • 依據網頁表單，設計 JSON 資料物件(如 moodle 課程專題討論區範例，可約助教或老師討論 JSON 資料規格)
  • 取得將表單輸入數值，將其轉換到 JSON 資料物件當中，將物件轉換成 JSON 結構化字串，並透過 HTTP 將字串資料上傳到伺服器端接收程式
  • 設計網頁，HTTP 取回 JSON 資料，並將 JSON資料表格呈現；每一列呈現一筆表單輸入資訊(若表單填寫資料眾多，可選重要欄位呈現即可

註: 作業繳交時，提供表單資料上傳，以及下載 JSON 資料表格呈現兩個網頁連結。 或提供一 index.html 網頁連結，再由 index.html 連到上傳及下載資料網頁 範例一 範例二
